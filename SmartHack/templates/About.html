Process of implementation

Our project uses the following technologies: Python, Firebase, HTML, CSS.
The web server receives the user request and passes it to the Flask application.
The Flask application uses News API to get the news. It classifies them as
negative, neutral or positive and sends all of them to the Firebase database
and shows only the 10 most recent back to the user application.
